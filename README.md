# GOT Character Database

Ce petit projet affiche une galerie de personnages de Game of Thrones à partir de l'API publique [Thrones API](https://thronesapi.com/).

## Fonctionnalités

- Affichage de personnages sous forme de cartes
- Recherche par nom
- Filtrage par maison/famille
- Interface simple et responsive

## Structure du projet

- `index.html` : structure HTML de la page
- `index.ts` : logique TypeScript de l’application
- `index.js` : version compilée / exécutée dans le navigateur

## Comment lancer le projet

1. Ouvrez le fichier `index.html` dans votre navigateur.
2. Si vous souhaitez modifier le code TypeScript, vous pouvez le compiler ensuite vers `index.js`.

## Technologies utilisées

- HTML
- CSS
- JavaScript
- TypeScript

## Notes

Ce projet est volontairement simple et sert de base pour découvrir la manipulation d’API et le filtrage dynamique de données.
